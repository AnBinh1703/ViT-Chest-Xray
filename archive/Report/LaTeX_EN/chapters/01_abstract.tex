\section*{Paper Summary}

This report provides an expert-level analysis of the paper \textit{``A Comparative Study of CNN, ResNet, and Vision Transformers for Multi-Classification of Chest Diseases''} (arXiv:2406.00237).

The paper compares 5 deep learning architectures for classifying 15 types of chest diseases from X-ray images:
\begin{enumerate}
    \item \textbf{CNN:} Baseline Convolutional Neural Network
    \item \textbf{ResNet-34:} Residual Network with 34 layers
    \item \textbf{ViT-v1/32:} Vision Transformer with 32×32 patch size (config 1)
    \item \textbf{ViT-v2/32:} Vision Transformer with 32×32 patch size (config 2)
    \item \textbf{ViT-ResNet/16:} Hybrid Vision Transformer with ResNet backbone
\end{enumerate}
